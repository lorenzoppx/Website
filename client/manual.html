Navegação


1. Setas que permitem as ações de back/recuar e forward/avançar em relação as páginas.
2. Botão para recarregar a página.
3. Botões que redirecionam para as páginas para enviar ou pesquisar os datasets.
Em caso de dúvida contate o desenvolvedor.

Passo a passo página de envio


1. Para usar a página de envio basta possuir um arquivo em formato .csv válido.
2. Ao clicar em buscar o cliente seleciona o arquivo de coleta desejado.
3. O nome do dataset tem que seguir o seguinte padrão: "Projeto-Dia/Mes". Exemplo: "ICEPI-03/02"
4. Após o término do upload a página indicará 100%.
5. Caso bem sucedido não aparecerá mensagens de erro.
Em caso de dúvida contate o desenvolvedor.

Passo a passo página de pesquisa


1. Para usar a página de envio basta procurar pelos datasets desejados, pelo projeto e/ou data. Exemplo: "ICEPI","03/02","ICEPI-03/02"
2. Após selecionar o dataset desejado, o sistema te redirecionará para a página de download.
3. A janela de busca possui setas navegação pelo tweets e o botão de download.
4. Após clicar no botão de download, o sistema te redirecionará para outra página, essa página NÃO pode ser fechada, após o processamento devido seu download é iniciado.
Em caso de dúvida contate o desenvolvedor.
